Spring Web MVC

Spring Web MVC is a Java based framework that facilitates web development through the use of the model-view-controller pattern.

Spring Web MVC features

This sample application demonstrates the use of the following features of the Spring Web MVC framework:

  1. Templating
    1. PHP
  2. Form binding
    1. Binding Java enumerations
  3. REST calls
    1. Data representations
    2. Object graph
    3. Generics
