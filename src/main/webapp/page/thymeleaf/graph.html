REST APIs and Object Graphs

Objects being serialized over a REST endpoint may contain references to other objects. It is frequently required to serialize the referenced objects as well. This sample uses two classed named Company and Project with a one-to-many association from the former to the later.

JSON representation

To see a sample object graph represented as JSON make a regular HTTP GET request to /graph.json on this server.

Since the controller for this example returns a collection of objects, there is no XML representation for the returned data, as there is no XML root tag for a collection element.
