Form Binding with Enumerations

Form Binding is a useful feature of Spring MVC that allows user-provided input to be dynamically bound to Java objects used for processing that input. The most common use of this feature is in mapping information provided in HTML forms using form controls to method parameters or object fields.

Some times user input needs to be bound to a field whose Java data type is an enumeration. This example uses a Java enumeration enum Day { SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY } and binds the selected value of an HTML radio button group to a Java variable of the type Day.
